Class Project

For our class on software engineering, we were tasked with developing an app, website, or application that would be useful to the public. In our group, we decided it would be helpful to have some kind of application to filter for allergens at websites. Based on previous experience, there is not much software out there with this specific functionality.

Motivations

Many people today have allergies that need to be avoided. With this in mind, it can be generally hard to locate allergen friendly places We wanted to create a website easy to use and easy to locate food options based on restaurants in the Indianola area.
